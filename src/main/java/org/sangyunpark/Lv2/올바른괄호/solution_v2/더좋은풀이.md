```java
    boolean solution(String s) {
        int answer = 0;
        
        for(char value : s.toCharArray()) {
            answer = value == ')' ? answer-1 : answer + 1;
            if(answer < 0) return false;
        }

        return answer == 0;
    }
```
내가 풀던 방식과 시간 복잡도는 동일하지만, 공간 복잡도 측면에서 훨씬 유리한 풀이이다.
이 풀이는 공간 복잡도가 O(1) 밖에 되지 않는다.
