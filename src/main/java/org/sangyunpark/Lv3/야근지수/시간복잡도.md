```java
public long solution(int n, int[] works) {
        long answer = 0;

        PriorityQueue<Long> pq = new PriorityQueue<>(Collections.reverseOrder());

        for(int i = 0; i < works.length; i++) { // O(n)
            pq.offer((long) works[i]); // O(logn)
        }

        for(int i = 0; i < n; i++) { // O(n)
            if(pq.isEmpty()) return 0; // 남은 작업량이 없는 경우
            long value = pq.poll(); // O(logn)
            if(value > 0) {
                pq.offer(value - 1);
            }
            // value가 0이면, poll해주고 다시 offer를 해주지 않는다.
        }

        while(!pq.isEmpty()) { // O(n)
            long value = pq.poll();
            answer += value * value;
        }

        return answer;
    }
```

offer() : O(logn)
for(i = 0) : O(n)

최종 시간 복잡도 : O(nlogn)
